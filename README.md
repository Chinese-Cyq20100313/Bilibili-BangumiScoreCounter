# Bilibili-BangumiScoreCounter
获取 真实的 B站番剧评分

## 使用方法  
> git clone https://github.com/Chinese-Cyq20100313/Bilibili-BangumiScoreCounter.git sc_counter  
cd sc_counter  
python ./main.py  

打开番剧，复制链接，粘贴。  
等待获取评价信息，完成
